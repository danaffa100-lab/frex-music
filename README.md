# Frex Music

Музыкальный сервис в визуальном стиле Яндекс Музыки: жёлтая тема, анимации «река и волна», плейлисты, подписка и админ-панель.

## Возможности

- Регистрация и вход
- Превью **20 секунд** без подписки
- Полное прослушивание и **«Мой плейлист»** с подпиской
- Плейлисты: Волна хитов, Река звуков и др.
- Админ **pax** / **Danil.228**: загрузка треков (аудио + обложка + название), блокировка треков и пользователей, выдача подписки

## Запуск

Требуется **Node.js 22+** (у вас 24 — подходит). База SQLite встроена в Node, без `better-sqlite3`.

```bash
npm install
npm run init-db
npm start
```

Главная страница: `index.html` в корне проекта (открывается через `http://localhost:3000`).

Сайт: http://localhost:3000
